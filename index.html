CodeHub

The most advanced client for GitHub on iOS!

About

CodeHub is the most complete, unofficial, client for GitHub on the iOS platform. The project is designed in C# using the Xamarin.iOS product with the expectation that it will one day operate on all major mobile platforms. CodeHub is not, in any way, affiliated with GitHub, Inc.

Open Source

The code is available on GitHub. You should definitely check it out.

User Driven

Support is given to those who ask! Leave a feature request at UserVoice or GitHub.

Common Code

Shares the same codebase as CodeBucket so feature improvements in one help the other!

Full Featured

CodeHub strives to have all the features GitHub has plus more! Check out a few below!

Issues & Pulls Made Easy

Management Star!

Create, read, and update your issues with ease.

Pull Requests

Pull Request viewing, commenting, diffing, and merging all included.

Markdown Approved

Read issues the way they're ment to be: with full markdown support including images!

Filter Away

Filter your issues, repositories, source code, and more, any way that works for you with a custom filter full of features.

Notifications & Events Be Aware

Notify Yourself

Scan through your notifications to keep yourself aware of commits, pull requests, and issues.

News & Events

Your news and events are one slide away at all times - for your account and any organization you belong to.

Changesets

Browse through any repository's commit log - viewing diffs and commenting on changes.

Push Notifications

Like to be informed immediately? Push notifications give you instant updates.

Repos & Code As GitHub Intended

Source Code

View branches, tags, and source code with beautiful syntax highlighting - even pdfs supported!

Manage it all

Star and watch repositories - even pin them to the slideout menu for quick navigation!

Gistacular

View personal, starred, and public gists. Even star new ones and share from within the app.

Explore Away

Search and explore new repositories to add to your ever growing collection of great code.
